# 功能
## 服务端
1. 使用node fastify+static web 实现一个http服务器，展示某个目录里面mp4视频，提供mp4文件上传，下载，删除，播放的接口，文件列表等，供web端调用。
2. 使用openapi 3.0 规范，生成接口文档。
3. 提供一个将视频传给fpga的接口，该接口接收一个视频文件路径，然后将视频通过pcie传输到fpga,传导fpga的功能先不实现，

## 客户端
1. 使用vue3+vite+element-plus 实现一个web端，调用服务端的接口，实现以下功能：
2. 展示服务端的视频列表，包括视频名称，视频大小，视频时长等；
3. 列表里面有上传，删除，播放的按钮，点击上传按钮，将视频上传到服务端；点击删除按钮，将视频从服务端删除；点击播放按钮，将视频通过网页播放。
4. 列表里面有一个按钮："传输到FPGA"，点击按钮，将选中的单个视频传给服务器，服务器将视频通过pcie传输到fpga

## 要求
1. 服务端使用nodejs开发
2. web代码放到 src/web/ 目录下面
3. 文件上传功能，你自己选择合适技术实现，要考虑大文件上传，视频文件可以大于2G
4. 开发完成后测试所有功能点，并提供测试报告
5. 测试完成后提push request
